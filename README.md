# Bach Chorale Harmonisation Trainer

An ML-powered pedagogical tool that helps A-Level music students practice Bach-style chorale harmonisation through real-time feedback and suggested solutions. 

## Overview

This project combines music theory education with machine learning to create an interactive learning environment for students studying Bach chorale harmonisation. The tool provides real-time feedback on student harmonisations, suggests improvements based on historical Bach chorales, and offers progressive learning exercises.

## Features

- **Real-time Harmonisation Analysis**: Analyzes student-written chorales against Bach's style
- **Interactive Exercises**: Progressive difficulty levels from basic chord progressions to full chorales
- **Style Compliance Checker**: Flags common stylistic errors (parallel fifths/octaves, voice leading issues)
- **ML-Powered Suggestions**: Provides AI-generated improvement recommendations based on trained models
- **Visual Score Display**: Renders chorales in standard musical notation using music21
- **Progress Tracking**: Monitors student improvement across exercises
- **Historical Reference**: Access to a database of actual Bach chorales for comparison
